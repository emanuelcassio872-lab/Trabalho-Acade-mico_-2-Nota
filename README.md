# Sistema de Oficina Mecânica

Projeto de Banco de Dados com Python + PostgreSQL.

## Funcionalidades
- Login
- CRUD de clientes
- Consulta com JOIN

## Tecnologias
- Python
- PostgreSQL
- psycopg2

## Como executar
1. Criar banco no PostgreSQL
2. Rodar scripts SQL
3. Executar main.py
